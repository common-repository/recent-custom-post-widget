This plugin display your recent custom post(s).

== Description ==

This plugin display your recent custom post(s). Visit -> Widgets and drag the recent custom post button into your sidebar.

== Installation ==

1. Download the plugin in your computer and upload it on your server /wp-content/plugins/
2. Activate the plugin from the wp admin.
3. To activate the plugin simply visit the widgets page and drag the recent custom post button in your sidebar.
